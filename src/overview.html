open-dis7-java provides a complete type-safe Java implementation of the DIS Protocol version 7 (IEEE 1278.1-2012) and SISO-REF-010 Enumerations specifications. interfaces and objects, all as open source.

Over 22,000 SISO-REF-010 enumeration values are provided. IEEE DIS Protocol Data Unit (PDU) packet-definition classes are defined in the edu.nps.moves.dis7.pdus package.

Further examples, presentations and projects are provided in Networked Graphics MV3500 course archive by the Modeling, Virtual Environments, Simulation (MOVES) Institute of the Naval Postgraduate School (NPS).
